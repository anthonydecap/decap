# MIDI Slice

A comprehensive slice for showcasing MIDI technology features, specifications, and performance statistics.

## Features

- **Title and Description**: Main heading with eyebrow text and rich text description
- **Main Image**: Large hero image to showcase the MIDI technology
- **Feature Grid**: Configurable features with icons and descriptions
- **Statistics Block**: Performance metrics and technical specifications
- **Subscript**: Additional technical details and explanations
- **Theme Support**: Light and dark theme options

## Usage

This slice is perfect for:
- MIDI technology product pages
- Technical specification showcases
- Performance comparison sections
- Feature highlight sections

## Configuration

### Primary Fields
- `title`: Main section title
- `eyebrow`: Small text above the title
- `description`: Rich text description
- `main_image`: Hero image for the section
- `invert`: Toggle between light and dark themes
- `stats`: Array of statistics with values, labels, and descriptions
- `subscript`: Additional technical information

### Items (Features)
- `feature_title`: Title of the feature
- `feature_description`: Rich text description of the feature
- `feature_icon`: Icon selection from predefined MIDI-related icons

## Icons Available
- `speed`: Lightning bolt for speed-related features
- `bandwidth`: Waveform for bandwidth/throughput
- `connector`: Connection/interface icons
- `precision`: Shield/checkmark for precision/accuracy
- `compatibility`: Heart for compatibility features
- `control`: Settings gear for control features
- `protocol`: Circle with checkmark for protocol standards
- `none`: No icon
