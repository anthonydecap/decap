# Possibilities Slice

A creative slice that showcases different possibilities with big, bold titles and engaging descriptions in various layout styles.

## Features

- **Creative Design**: Eye-catching blocks with gradient accents and hover effects
- **Big Typography**: Large, bold titles that make a statement
- **Multiple Layouts**: Grid, masonry, and stacked layout options
- **Color Customization**: 6 accent colors for each possibility block
- **Icon Support**: Emoji or symbol icons for visual appeal
- **Responsive Design**: Adapts beautifully to all screen sizes
- **Smooth Animations**: Staggered fade-in effects and hover interactions

## Configuration

### Primary Fields

- **Section Title**: Large heading for the entire section
- **Section Subtitle**: Description text for the section
- **Layout Style**: Choose between grid, masonry, or stacked layouts
- **Background Color**: Select white, light, or dark background

### Possibility Items

Each possibility block can have:
- **Possibility Title**: Big, creative title (3xl to 5xl font size)
- **Possibility Description**: Rich text description with formatting
- **Accent Color**: Choose from 6 color options (blue, green, purple, orange, red, yellow)
- **Icon**: Emoji or symbol for visual enhancement

## Layout Styles

### Grid Layout (Default)
- Clean 3-column grid on desktop
- Responsive: 1 column on mobile, 2 on tablet, 3 on desktop
- Equal height cards with consistent spacing

### Masonry Layout
- Pinterest-style masonry layout
- Cards flow naturally based on content height
- Creates dynamic, organic feel

### Stacked Layout
- Single column layout with centered cards
- Maximum width for optimal readability
- Perfect for storytelling or sequential content

## Accent Colors

Each possibility block can have a unique accent color:
- **Blue**: Professional and trustworthy
- **Green**: Growth and sustainability
- **Purple**: Innovation and creativity
- **Orange**: Energy and enthusiasm
- **Red**: Power and urgency
- **Yellow**: Optimism and clarity

## Styling Features

- **Gradient Accent Lines**: Colorful top borders for each block
- **Hover Effects**: Scale animation and subtle overlay on hover
- **Shadow System**: Elevation with shadows and hover shadows
- **Rounded Corners**: Modern 3xl border radius
- **Typography**: Large, bold titles with readable descriptions
- **Icon Containers**: Rounded backgrounds for icons

## Usage

1. Add the Possibilities slice to your page in Prismic
2. Configure the section title and subtitle
3. Choose layout style and background color
4. Add possibility blocks by clicking "Add Item"
5. For each possibility:
   - Add a big, creative title
   - Write an engaging description
   - Select an accent color
   - Add an icon (emoji or symbol)
6. The possibilities will display in your chosen layout with smooth animations

## Best Practices

- **Titles**: Keep them short and impactful (2-4 words)
- **Descriptions**: Write compelling, benefit-focused content
- **Icons**: Use relevant emojis or symbols that match the content
- **Colors**: Mix and match colors for visual variety
- **Layout**: Choose layout based on content length and visual goals

## Accessibility

- Semantic HTML structure
- Proper heading hierarchy
- High contrast text and backgrounds
- Keyboard navigation support
- Screen reader friendly
- Responsive design for all devices
