# SmartValveVideo

Highlighted YouTube video section with title and description. SmartValve styling (dark background, centered content).

## Fields

- **Title** – StructuredText (heading)
- **Description** – StructuredText (paragraph, links, etc.)
- **YouTube URL or Video ID** – Paste full URL (`https://www.youtube.com/watch?v=...` or `https://youtu.be/...`) or just the 11-character video ID
